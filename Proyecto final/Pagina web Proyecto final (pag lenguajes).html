Herramientas de Desarrollo

  • Visual Studio Code: Un editor de código ligero y potente desarrollado por Microsoft, con soporte para una variedad de lenguajes y extensiones.
  • Git: Un sistema de control de versiones distribuido ampliamente utilizado para el seguimiento de cambios en el código fuente durante el desarrollo de software.
  • PyCharm: Un entorno de desarrollo integrado (IDE) específico para Python, que ofrece características avanzadas para el desarrollo web, científico y de datos.
  • Node.js: Un entorno de tiempo de ejecución de JavaScript basado en el motor V8 de Chrome, que permite ejecutar JavaScript en el lado del servidor y construir aplicaciones web escalables.
  • React: Una biblioteca de JavaScript para construir interfaces de usuario interactivas y reutilizables, desarrollada por Facebook.
